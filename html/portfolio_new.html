📌 Summary

기존의 포트폴리오를 갈아엎어 완전히 새로 만들어낸 포트폴리오 사이트입니다.

짜임새를 약간 수정하고, 조금 더 깔끔하게 보일 수 있도록 디자인을 완전히 변경하였습니다.
Mobile / Tablet과 Desktop의 디자인을 조금 더 명확하게 구분할 수 있게 사이드바의 위치를 조정함으로서 구분감을 주었고, Projects의 구분을 편하게 하기 위해 Data Filter 기능을 활용하여, 조금 더 사용자 편의적인 디자인고가 기능을 구현할 수 있도록 노력했습니다.

프론트는 기존의 포트폴리오와 마찬가지로 HTML5, CSS3, Vanilla JS를 사용하였으며, GitHub Pages 기능을 활용하여 Deploy 했습니다.
