📌 Summary

기존에 제작해둔 포트폴리오 사이트입니다.

전체적인 짜임새는 괜찮게 짰다고 생각했으나, Desktop 크기로 보는데도 불구하고 Tablet 상의 화면을 늘린 느낌이 나기도 하고, Background의 색상으로 지정해둔 Gradient 효과가 너무 튀어보이는 것 같아 짜임새는 일부분 가져가고 디자인 자체를 완전히 갈아엎기로 했습니다.

HTML5, CSS3, Vanilla JS로 프론트를 작성해보며, 세 언어의 기초를 다시 한번 복습할 수 있었고, GitHub 내에서 제공해주는 Pages 서비스를 처음으로 사용해보며, 간단한 사이트같은 경우에는 GitHub 내의 Pages 서비스를 이용해도 될 만큼 괜찮은 성능에 감탄하며 새로운 지식들을 알게 되어 개발하는 과정 자체가 매우 재미있고 흥미로웠습니다.
